Virtual Mathematics Laboratory – Exact Differential Equations

Aim

To study the concept of exact and non-exact differential equations and to develop an interactive virtual laboratory that determines the exactness of a given first-order differential equation of the form M(x,y)dx + N(x,y)dy = 0. The experiment also aims to help students understand the role of integrating factors and the systematic procedure for obtaining the general solution.

Theory

A first-order differential equation of the form M(x,y)dx + N(x,y)dy = 0 is said to be an exact differential equation if there exists a scalar function F(x,y) such that dF = Mdx + Ndy. This implies that M = ∂F/∂x and N = ∂F/∂y. A necessary and sufficient condition for exactness is that the mixed partial derivatives are equal, i.e., ∂M/∂y = ∂N/∂x, provided the functions are continuous in the given region.

If the given equation is exact, the solution can be obtained by integrating M with respect to x (treating y as constant) to obtain F, and then determining the remaining function of y by comparing with N. The final solution is expressed implicitly as F(x,y) = C.

Many practical differential equations are not exact in their original form. In such cases, an integrating factor μ may be introduced to convert the equation into an exact one. If (∂M/∂y − ∂N/∂x)/N is a function of x alone, then an integrating factor μ(x) exists. Similarly, if (∂N/∂x − ∂M/∂y)/M is a function of y alone, then μ(y) exists. Multiplying the original equation by the appropriate integrating factor transforms it into an exact equation, which can then be solved using the standard procedure. Exact differential equations appear frequently in thermodynamics, fluid mechanics, and electrical engineering.

Pre-Test

1. Which condition defines an exact differential equation?

a) ∂M/∂x = ∂N/∂y
b) ∂M/∂y = ∂N/∂x (Correct)
c) M = N
d) dy/dx = M/N

2. For the equation (2x+y)dx + (x+2y)dy = 0, is it exact?

a) Yes
b) No (Correct, since ∂M/∂y = 1 ≠ 2 = ∂N/∂x)

3. Integrating factor depending only on x satisfies:

a) (∂M/∂y − ∂N/∂x)/N = f(x) (Correct)

Procedure

  1. Input the functions M(x,y) and N(x,y).
  2. Compute ∂M/∂y and ∂N/∂x.
  3. If ∂M/∂y = ∂N/∂x, the equation is exact.
  4. Integrate M with respect to x to obtain F(x,y).
  5. Add the missing function of y by comparing with N.
  6. If non-exact, test for integrating factor μ(x) using (∂M/∂y − ∂N/∂x)/N.
  7. Similarly test μ(y) using (∂N/∂x − ∂M/∂y)/M.
  8. Multiply by μ and solve as an exact equation.
  9. Display the solution F(x,y) = C.
